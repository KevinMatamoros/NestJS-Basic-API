## Description

BASIC API - CRUD of Messages.
Example of object created


```
{
  nick: 'Kevin M'
  message: 'How are you doing?'
}
```

## Installation

```bash
$ npm install
```

## Script to generate DB

File: `src/connections.ts` contain data of connection to dabase, just is needed create a database in XAMPP with that specification to can run the app.

Note: `NOT NEEDED` but
The file called `sendmeapp_db.sql` contain the script to generate DB.


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
